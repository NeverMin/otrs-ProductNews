=head1 NAME

ProductNews - Show news about the product on the Dashboard

=head1 DESCRIPTION

If you have news about your product or any other message that all agents of your
OTRS system should know about, you can use ProductNews to handle this.

With this extension a new dashboard widget is created and administrators can add
news.

=head1 REPOSITORY AND BUGTRACKER

The code repository and a bugtracker is available at L<http://github.com/reneeb/otrs-ProductNews>.
